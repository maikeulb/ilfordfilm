# IlfordFilms

Frontend for an E-commerce application where users can browse a catalog of
Ilford 35mm films, add films to their shopping cart, submit orders, and review their
order history. 

Technology
----------
* React
* Redux + Redux-Thunk
* Firebase
* Ant-Design
* Styled-Components

Infrastructure
----------
* Firebase
